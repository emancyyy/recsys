# Recsys Project — Recommendation System (PHP + JS)


Полный рабочий проект рекомендательной системы со следующими функциями:
- Автоматическая генерация данных (сидов)
- Рекомендации на основе тегов и лайков
- Логика совпадения тегов (чем больше совпадений — тем выше пост)
- 30 постов с расширенными тегами
- 5 пользователей
- Интеракции (likes / views)
- Красивый тёмный интерфейс 
- 5 постов на странице


---


## Функционал


### Рекомендации
Алгоритм рекомендаций строится по количеству совпадающих тегов.
Посты сортируются по:
1. Количество совпавших тегов (DESC)
2. Популярность (количество interactions)
3. Новизна (recency bonus)
4. ID


### Лайки
Пользователь вводит user_id и может:
- Ставить / убирать лайки
- Система пересчитывает предпочтения
- Рекомендации обновляются автоматически
